Python图像处理

发表于 2017-12-20 | 分类于 Python

1.颜色与RGBA值：

计算机通常将图像表示为RGB值，或者再加上alpha值（通透度，透明度），称为RGBA值。在Pillow中，RGBA的值表示为由4个整数组成的元组，分别是R、G、B、A。整数的范围0~255。RGB全0就可以表示黑色，全255代表黑色。可以猜测(255, 0, 0, 255)代表红色，因为R分量最大，G、B分量为0，所以呈现出来是红色。但是当alpha值为0时，无论是什么颜色，该颜色都不可见，可以理解为透明。

2.图像的坐标表示

图像中左上角是坐标原点(0, 0)，X轴是从左到右增长的，而Y轴是从上到下增长。在Pillow中使用包含四个值的元组表示一个矩形区域，表示顺序是(左，顶，右，底)。右和底坐标稍微特殊，表示直到但不包括。可以理解为[左, 右)和[顶， 底)这样左闭右开的区间。比如(3, 2, 8, 9)就表示了横坐标范围[3, 7]；纵坐标范围[2, 8]的矩形区域。

3.使用pillow操作图像

openCV中是imread(),pillow中是open()。

im.size返回一个元组，分别是宽和高。show()方法会调用系统默认图像查看软件，打开并显示。im.format可查看图像的格式。save()可保存处理后的图片，如果未经处理，保存后的图像占用的空间(字节数)一般也与原图像不一样，可能经过了压缩。

4.新建图像

Image.new()函数可以用来新建图像，第一个参数是mode即颜色空间模式，第二个参数指定了图像的分辨率(宽x高)，第三个参数是颜色。
可以直接填入常用颜色的名称。如’red’ 也可以填入十六进制表示的颜色，如#FF0000表示红色。
*还能传入元组，比如(255, 0, 0, 255)或者(255， 0， 0)表示红色。
